# Learning Markdown

Markdown has 6 level of headings. Corresponding to HTML tags h1,h2,...,h6.

# H1
## H2
### H3
#### H4
##### H5
###### H6
####### H7 This is not heading. Follow this guide carefully to learn markdown.

Styling in markdown -
- **bold**
- *italics*
- ~~strikethrough~~
